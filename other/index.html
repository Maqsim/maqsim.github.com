Підтримувані функції калькулятора

Константи

  • e — основа натурального логарифма з наближеним числовим значенням 2.71828...
  • pi — число, що має значення 3.14159... і рівне відношенню довжини кола до його діаметру
  • i — представляє уявну одиницю, sqrt(-1)
  • Degree — число радіан в одному градусі, яке має числове значення pi/180
  • EulerGamma — постійна Ейлера з числовим значенням 0.577216...
  • GoldenRatio — константа зі значенням (1 + sqrt (5)) / 2, визначаюча поділ відрізка за правилом золотого перетину

Елементарні функції

  • abs(x) — модуль значення x, |x|
  • sqrt(x) — квадратний корінь значення x, √x
  • x^y — x в степені y
  • exp(x) — экспонента значення x, e^x
  • a logb — логарифм значення b за основою a, Log(a,b)
  • ln x — натуральний логарифм значення x, Log(e,x)
  • n! — факторіал числа n

Тригонометричні функції

  • sin(x) — синус значення x
  • cos(x) — косинус значення x
  • tan(x) — тангенс значення x
  • cot(x) — котангенс значення x
  • sec(x) — секанс значення x, sec(x)=1/cos(x)
  • csc(x) — косеканс значення x, csc(x)=1/sin(x)

Зворотні тригонометричні функції

  • sinh(x) — синус гіперболічний значення x
  • cosh(x) — косинус гіперболічний значення x
  • tanh(x) — тангенс гіперболічний значення x
  • coth(x) — котангенc гіперболічний значення x
  • sech(x) — секанс гіперболічний значення x
  • csch(x) — косеканс гіперболічний значення x

Гіперболічні функції

  • arcsinh(x) — арксинус гіперболічний значення x
  • arccosh(x) — арккосинус гіперболічний значення x
  • arctanh(x) — арктангенс гіперболічний значення x
  • arccoth(x) — арккотангенc гіперболічний значення x
  • arcsech(x) — арксеканс гіперболічний значення x
  • arccsch(x) — арккосеканс гіперболічний значення x
